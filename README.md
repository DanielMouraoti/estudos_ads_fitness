# estudos_ads_fitness

Projeto de estudos _Análise e Desenvolvimento de Sistemas_ (ADS) com foco em automação e geração de treinos para a área de fitness.

## 🏋️ Sobre o projeto

Este repositório reúne scripts, exemplos práticos e ferramentas criados para facilitar estudos e experimentos sobre automação, programação Python e aplicações relacionadas ao mundo fitness (rotinas de treino, gerações de fichas, cálculos, etc).

O objetivo é apoiar o aprendizado de conceitos de desenvolvimento de software enquanto resolve problemas reais ou simulados do universo fitness, como geração automatizada de treinos, cálculo de IMC, integração com APIs, manipulação de dados e muito mais.

## 🚀 Como rodar os scripts

1. **Pré-requisitos:**  
   Tenha o [Python 3.8+](https://www.python.org/downloads/) instalado em seu sistema.

2. **Clone o repositório:**
   ```bash
   git clone https://github.com/DanielMouraoti/estudos_ads_fitness.git
   cd estudos_ads_fitness
   ```

3. **Instale dependências (caso o projeto tenha):**
   ```bash
   pip install -r requirements.txt
   ```
   > Caso não exista um arquivo `requirements.txt`, os scripts podem rodar sem instalação extra.

4. **Rode um dos scripts:**
   ```bash
   python3 gerador_treino.py
   ```
   *(Adapte esse comando de acordo com o nome do arquivo ou script que deseja executar)*
